To clone a git repo using python, consider using package GitPython.

To install the package, run

pip install GitPython

It will install the latest version of the package. Refer the pypi org for complete documentation

from git import Repo
path_to_pull = "/home/user/tmp"
repo = Repo.init(path_to_pull)
repo.create_remote('origin', arl=<remote url>)
repo.remote().fetch()
repo.git.pull()

Simply, we are initialize the directory, create remote using remote url, fetching the commits and ref and pull it.
